- What it is
  This is a program dedicated to helping store information and run a website for a gym. It keeps track of staff and members and allows them to both have their own pages for navigating their own information.
  
- How to use
  To use this you run the main.py file. Here you can either choose to sign in as a staff or member. If you have a log in you can get in, otherwise you must have the manager add a login if you are new staff or have a staff add you if you are a new member.
