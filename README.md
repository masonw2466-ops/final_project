- What it is

This is a program dedicated to helping store information and run a website for a gym. It keeps track of staff and members and allows them to both have their own pages for navigating their own information.
  
- How to use

To use this you run the main.py file. Here you can either choose to sign in as a staff or member. If you have a log in you can get in, otherwise you must have the manager add a login if you are new staff or have a staff add you if you are a new member.

- Usage example

1. Run the program: python3 main.py
2.	Choose your role:
	•	1. Staff Login
	•	2. Member Login
3.	Enter your username and password.
4.	If you are staff, you can:
	•	Add new members
	•	Add new staff accounts (manager only)
	•	View or update member information
	•	Manage gym records
5.	If you are a member, you can:
	•	View your personal account information
	•	Check membership status
	•	Update your profile
6.	Log out when finished.

- Dependancies

For this program you must have:
1. tkinter
2. sqlite3
